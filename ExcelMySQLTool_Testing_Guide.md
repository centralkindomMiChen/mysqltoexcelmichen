# ExcelMySQLTool - Testing Guide

## 1. Introduction

### Purpose
This document provides a comprehensive testing guide for the "ExcelMySQLTool" application. It outlines the test cases designed to verify the application's functionality, UTF-8 character handling, and compatibility with the specified environment (Windows 7 64-bit, MySQL 5.4).

### Application Overview
ExcelMySQLTool is a Windows Forms application that facilitates data transfer between Microsoft Excel files (`.xls` and `.xlsx`) and a MySQL database. Key features include:
*   Importing data from Excel files into a MySQL table.
*   Exporting data from a MySQL table to Excel or CSV files.
*   Previewing data from Excel files and MySQL tables.
*   Support for UTF-8 character encoding, particularly for handling languages like Chinese.
*   Database connection configuration.
*   Filtering exported data by a date range.

## 2. Prerequisites for Testing

*   **Operating System:** Windows 7 (64-bit) environment (physical machine or a Virtual Machine).
*   **MySQL Server:**
    *   MySQL version 5.4 instance accessible over the network or locally.
    *   A dedicated test database created on this server (e.g., `excel_mysql_test_db`).
    *   MySQL user credentials with full permissions (CREATE, INSERT, SELECT, DROP TABLE) on the test database.
*   **.NET Framework:** .NET Framework 4.5 (or the specific version targeted by the final compiled application) must be installed on the Windows 7 machine.
*   **Sample Excel Files:** A collection of `.xls` and `.xlsx` files, including:
    *   `SimpleData.xlsx`, `SimpleData.xls`: Containing various data types (text, numbers, dates without special characters).
    *   `ChineseData.xlsx`, `ChineseData.xls`: Containing Chinese characters in cell data.
    *   `ChineseHeaders.xlsx`, `ChineseHeaders.xls`: Containing Chinese characters in column headers.
    *   `ChineseFileName_测试.xlsx`, `ChineseFileName_测试.xls`: Excel files with Chinese characters in their actual file names.
    *   `DateFilterData.xlsx`, `DateFilterData.xls`: Files with a clearly defined date column (e.g., "OrderDate") and data spanning several months/years for testing date range export.
    *   `EmptySheet.xlsx`, `EmptySheet.xls`: Files that are completely empty.
    *   `HeaderOnly.xlsx`, `HeaderOnly.xls`: Files that contain only a header row and no data rows.
    *   `MixedData.xlsx`: A file containing a mix of alphanumeric, special characters, and some Chinese characters.
*   **Application Executable:** The compiled `ExcelMySQLTool.exe` and any associated DLLs (like `MySql.Data.dll`, `EPPlus.dll`, `ExcelDataReader.dll`).
